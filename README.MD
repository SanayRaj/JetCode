## How to Contribute

### Clone this repo:

```sh
git clone https://github.com/SanayRaj/JetCode.git
```

### Install dependencies:

```sh
npm install
```

### Run app:
> This command run app in browser
```sh
npm run dev
```

#### To run app in physical device:
```sh
npmk run android
```

> Make sure a physical device is connected & detected in `adb` OR emulator is installed.
